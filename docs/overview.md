# Loopback

**ふりかえりのハードルを下げ、目標と日常のループを回すセルフマネジメントツール**

---

## 解決する課題

ふりかえりは面倒で続かない。続かないから目標と日常が乖離する。
次の目標設定のときには過去を思い出せず、的外れな目標を立ててしまう。

---

## コンセプト

**目標設定 → 高頻度ふりかえり → ログ蓄積 → 次の目標設定が的確に → …**

このループを回し続けることがLoopbackの核。

- **目標は先に立てる**（トップダウン）
- **ふりかえりを気軽に、高頻度で行う** — 日記・日報のように毎日でも苦にならない体験
- **気軽に振り返るから目標を忘れずアラインできる**
- **ログが溜まることで、次の目標設定がより適切になる**

AIはこのループを滑らかに回すための潤滑油。ふりかえりに伴走し、ログの活用を助ける。

> "振り返ることで、前に進む。"

---

## 特徴

- **ローカルファースト** — データはSQLiteでローカルファイルとして保持。クラウド不要
- **`loopback start` で即起動** — コマンド一発でWeb UI・DB・MCPサーバーが全て立ち上がる
- **シンプルでポップなUI** — ふりかえりのハードルを下げるための軽い体験
- **Claude Codeと連携** — Claude CodeやClaude Desktop AppからMCP経由でLoopbackのデータに接続し、ふりかえりに伴走できる
- **AIはあなたのサブスクを使う** — Loopback自体はAIを持たない。使い慣れたClaudeをそのまま使う
- **外部ツール連携** — Claude Code経由でNotionなど既存の自己管理ツールとも連携可能

## ターゲット

- Claude Code / Claude Desktop App を使うすべてのビジネスパーソン

---

## アーキテクチャ概要

```
[Web UI]  ← ブラウザからアクセス（シンプル・ポップ）
    |
[API layer]  ← REST API + ローカル MCP (SSE) エンドポイントを公開
    |
[SQLite]  ← ローカルファイルとしてデータ永続化

[Claude Code / Claude Desktop App]  ← MCP (SSE) 経由で接続
                                       Notion等の外部ツールとも連携可能
```
