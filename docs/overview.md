# Loopback

**週次ふりかえりを積み重ね、AIと伴走しながら成長を記録するセルフマネジメントツール**

---

## コンセプト

MBO的な年次・四半期目標を軸に、毎週のふりかえりを記録していく。
ログが蓄積されることで、自分の成長・変化・パターンが可視化される。
AIはユーザー自身のサブスクリプション（Claude Code / Codex等）を使い、
MCPまたはCLI経由でLoopbackのデータにアクセスして伴走する。

> "振り返ることで、前に進む。"

---

## 特徴

- **ローカルファースト** — データはSQLiteでローカルに保持。クラウド不要、プライバシー担保
- **Dockerで即起動** — `docker compose up` でWeb UIがすぐ使える
- **AIはあなたのサブスクを使う** — Loopback自体はAIを持たない。Claude Code / Codex等をMCP・CLI経由で接続
- **MCPサーバーとして公開** — Claude CodeなどのAIエージェントがLoopbackのデータを読み、ふりかえりに伴走できる
- **構造化されたふりかえりサイクル** — 週次 / 四半期 / 年次の3層で記録を管理

---

## ターゲット

- Claude Code / Codex などAIツールを日常的に使うエンジニア
- 目標管理・ふりかえりを習慣化したいビジネスパーソン

---

## アーキテクチャ概要

```
[Web UI]  ← Docker で起動、ブラウザからアクセス
    |
[API layer]  ← MCP サーバー / CLI エンドポイントを公開
    |
[SQLite]  ← ローカルにデータ永続化

[AI]  ← ユーザーの Claude Code / Codex 等が MCP 経由で接続
```
